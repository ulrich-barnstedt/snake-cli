## snake-cli

A simple snake implementation written in JS, designed for use in the terminal with Node.js

#### Usage

- Clone repo
- Open terminal
- Go to repo
- `git submodule update --remote --merge`
- `npm install`
- `node index.js`

#### Keys

- W/A/S/D or Up/Right/Down/Left - Movement
- R - Reset
